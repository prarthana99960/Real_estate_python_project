# Real_estate_python_project
Using Python libraries such as Pandas, Matplotlib, and Seaborn, the project answers multiple business-driven analytical questions related to real estate properties.
#  Real Estate Data Analysis Using Python

## 📌Project Overview

This project performs comprehensive data analysis on real estate property data using Python. The goal is to identify pricing trends, locality comparisons, builder performance, and other important insights from housing market data.

The project includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Business Insights
- Data Visualization

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn

---

## Dataset Features

The dataset contains property-related information such as:

- Price
- Area
- Rate per Square Foot
- Locality
- BHK Count
- Property Status
- RERA Approval
- Flat Type
- Builder/Company Name
- Society Name

---

## 📊 Key Business Questions Solved

### 1. Which is the costliest flat in the dataset?
Identifies the highest-priced property.

### 2. Which locality has the highest average price?
Finds premium real estate locations.

### 3. Which locality has the highest rate per square foot?
Determines the most expensive areas based on size pricing.

### 4. Do ready-to-move properties cost more?
Compares pricing between ready-to-move and under-construction properties.

### 5. Do RERA-approved properties command a price premium?
Analyzes the impact of RERA approval on property prices.

### 6. How does area impact price?
Visualizes the relationship between area and price.

### 7. Which BHK configuration is most expensive?
Finds the costliest BHK type based on rate per square foot.

### 8. Which property type is the most expensive?
Compares property categories using pricing metrics.

### 9. Which builders charge premium prices?
Identifies top premium builders based on average rate per square foot.

### 10. Are larger homes more expensive per square foot?
Analyzes area vs price-per-square-foot relationship.

## Data Cleaning Steps

- Removed duplicate records
- Standardized column names
- Cleaned numerical columns
- Converted price and area into numeric formats
- Normalized categorical values
- Converted RERA approval status into boolean values
